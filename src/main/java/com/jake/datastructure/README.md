### 자료구조
## 자바 컬렉션 프레임워크
- 비슷한 종류의 데이터들을 쉽게 다루기 위해 모아놓은 것들을 가종 및 처리할 수 있도록 지원
# 자료구조의 분류법
- 형태에 따른 자료구조 분류 
  - 선형 자료구조(Linear Data Structure)
    1. 리스트(List)
    2. 큐(Queue)
    3. 덱(Deque)
  - 비선형 자료구조(Nonlinear Data Structure)
    1. 그래프(Graph)
    2. 트리(Tree)
- 기타 자료구조
  - 집합 자료구조
    1. 집합(Set)
  - 파일 자료구조
    1. 순차파일
    2. 색인파일
    3. 직접파일
